# Role Access Matrix

## Admin

- Full create, read, update, delete, pause, and restore access for all modules.
- Can view all branches or filter by one branch.
- Can open complete student profile from student name.
- Can open complete staff profile from employee name.
- Can view feedback, performance, fleet data, camera evidence, financials, audit logs, and delete requests.
- Can approve or reject trainer leave.

## Receptionist

- Branch-limited access.
- Can manage enquiry, advance booking, admission, documents, internal RTO work, external RTO work, payment links, and receipts where permitted.
- Can view restricted student fields: name, address, education, occupation, photo, documents, LL number, LL date, payment work status, RTO work status, slot, and admission status.
- Cannot view trainer/student feedback, performance analytics, fleet data, camera data, admin-only evidence, or full earnings.
- Cannot directly delete records.

## Supervisor

- Branch-limited access.
- Can view restricted student data, leave students, DL practice list, DL test events, ready/not-ready students, and trainer leave application status.
- Can create DL test event dates and add ready students.
- Can update DL practice feedback and readiness.
- Cannot approve trainer leave.
- Cannot view fleet data, camera data, sensitive evidence, or admin financial reports.

## Trainer

- Can view assigned students and sessions.
- Can accept student session start requests.
- Can select the car used for a session.
- Can submit training feedback and car expenses.
- Can request leave.

## Student

- Can view own admission, payment, receipt, leave, training progress, videos, quiz, LL/DL status, and notifications.
- Can update own leave.
- Can request session start.
- Cannot view staff, other students, fleet evidence, or admin data.
